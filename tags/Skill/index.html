搭建个人github.io博客

环境要求

  • Node
  • Git
  • GitHub账户
  • Hexo

安装HEXO

  1. 打开终端，执行如下命令
    npm install -g hexo (win)
    sudo npm install -g hexo (unix)

  2. 创建文件夹
    mkdir blog

  3. 进入文件夹
    cd blog

  4. 项目初始化
    hexo init
