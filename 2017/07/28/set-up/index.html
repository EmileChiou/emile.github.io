搭建个人github.io博客

Veröffentlicht am 2017-07-28

环境要求

  • Node
  • Git
  • GitHub账户
  • Hexo

安装HEXO

  1. 打开终端，执行如下命令
    npm install -g hexo (win)
    sudo npm install -g hexo (unix)

  2. 创建文件夹
    mkdir blog

  3. 进入文件夹
    cd blog

  4. 项目初始化
    hexo init

  5. 生成静态页面
    hexo generate

  6. 生成静态页面(hexo版本3.3.8自带server)
    hexo server

  7. 打开浏览器，输入localhost:4000，即可进入博客

第一次部署到远端GitHub仓库

1.打开您的Github,选择Create New Repository创建一个新项目库
仓库名称命名格式必须为：你的github账户名称.github.io

  1. 配置vim _config.yml
    打开项目根目录的vim _config.yml。下拉到最后

改为如下

  1. 打开终端，执行如下命令，安装hexo git部署器
    npm install hexo-deployer-git --save

  2. 打开终端，执行如下命令，部署到github
    hexo deploy

  3. 打开浏览器，输入http://你的github账户名称.github.io/，即可进入博客
