长度单位的选择

  • 不能使用绝对宽度，只能指定百分比宽度、auto
  • 字体大小不能使用绝对大小（px），只能使用相对大小（em、rem）

宽度

  • width:视口宽度（视口指浏览器窗口内的内容区域，不包含工具栏，标签栏等）
  • height：视口高度
  • device-width:渲染表面的宽度（对我们来说，就是设备屏幕的宽度）
  • device-height:渲染表面的高度（对我们来说，就是设备屏幕的高度）
